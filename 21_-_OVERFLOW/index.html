CSS: Propriedade Overflow

A propriedade overflow especifica quando o conteúdo de um elemento de nível de bloco deve ser cortado, exibido com barras de rolagem ou se transborda do elemento. Esse comando apenas funciona se for aplicado uma altura ao elemento pai, dessa forma, o overflow encaixará o conteúdo ao elemento.


Exemplo 01 - VISIBLE (mostra o conteúdo mesmo que passe pra fora do elemento)




Exemplo 02 - HIDDEN (mostra apenas o conteúdo que se encaixa no elemento)




Exemplo 03 - SCROLL (aplica uma barra de rolagem para mostrar todo o conteúdo excedente)




Exemplo 04 - AUTO (aplica (somente se necessário) uma barra de rolagem para mostrar o conteúdo excedente)
