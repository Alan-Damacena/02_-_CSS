CSS: Seletores e Classes

Um seletor é um padrão de elementos e outros termos que informam ao navegador quais elementos HTML devem ser selecionados para que os valores de propriedade CSS dentro da regra sejam aplicados a eles

o ID é um seletor mais reservado uma vez que só pode ser usada por 1 único objeto na página.

Classes consiste em especificar uma tag para declaração de estilo
